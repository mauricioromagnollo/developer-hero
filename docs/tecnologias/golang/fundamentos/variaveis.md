# **Variables**

---

## **Const**

```go
const foo = "bar"
const foo = "bar"
```

## **Var**

As variáveis decalaradas em Go sem atribuição, possuem um valor inferido.

```go
var foo bool

foo = false
```

Declarando múltiplas variáveis.

```go
var (
	foo bool
	bar string
	baz int
)
```
